Notes

This app provides an interface for R package juicedown. If you require full control, please use juicedown directly.

The version of the app hosted on Shinyapps.io has further restrictions on executing arbitrary R code due to security reasons. This limitation is in place to prevent the risk of users running malicious code. If you prefer the user-friendly interface, run juicedownApp locally. See the installation section below.

In restricted mode, you still have access to a few app-only commands.

Install juicedown and juicedownApp

To install:

install.packages("remotes")
remotes::install_github("kenjisato/juicedownApp")

To run the app on the local server:

juicedownApp::up()
