Notes

This app provides an interface for R package juicedown. If you require full control, please use juicedown directly.

juicedown functions

juicedown has a few convenience functions.

YouTube Video

```{r}
includeYT("https://youtu.be/videoid")
```

Audio File

```{r}
includeAudio("https://example.com")
```

Quiz and Solution

```{r}
includeQuestion(
"What is the capital of Japan?",
#-----
"Tokyo",
#-----
summary_label = "Question", detail_label = "Answer")
```
Question

What is the capital of Japan?

Answer

Tokyo
